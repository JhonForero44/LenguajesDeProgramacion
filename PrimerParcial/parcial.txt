Enunciado:
ESTADÍSTICAS DE UN RPG

Como integrante de un equipo de desarrollo en un estudio de videojuegos, se le pide que implemente un módulo de análisis estadístico de personajes, y le dan como insumo un archivo plano CSV (comma separated values) con datos de algunos de ellos, con el siguiente modelo:

nombre,nivel,clase,ataque,defensa
Axxel,15,Guerrero,120,80
Luna,8,Mago,90,40
Drake,18,Arquero,140,70
Kai,12,Asesino,110,50
Zara,5,Mago,70,30
Nova,20,Guerrero,150,100
Bruce,17,Arquero,180,85
Debe procesar la información del dataset utilizando los conceptos de programación funcional en Python vistos en clase, e implementar los siguientes requerimientos:

lazy generator: Cargar los datos desde personajes.csv de forma diferida.
Funciones puras y lambdas: Implementar procesamiento sin efectos secundarios.
Funciones de orden superior: Utilizar map, filter y reduce.
Procedimiento:

Su programa debe cargar los datos desde el csv iterando sobre cada renglón y usando la función dict y zip. No se permite el uso de CSVReader o DictReader.
Cada registro del dataset debe ser un diccionario con los campos especificados en el encabezado del archivo plano.
El lector de datos debe devolver solo un registro a la vez y esperar a que lo consuman (lazy generator)
El análisis del dataset incluye:
filtrar personajes con nivel > 10 usando una función lambda
usar map con una función lambda para crear una nueva lista donde se tenga un campo adicional "totalPower", que se calcule como ataque + defensa
obtener el personaje con mayor poder total de todo el dataset usando reduce
Referencias:
